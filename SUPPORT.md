# Support

Use [GitHub Issues](https://github.com/coinsecuritiescompany/GPT-wallet-AiFinPay/issues) for reproducible, non-sensitive bugs and feature requests.

Include:

- platform and ChatGPT surface;
- approximate time and affected route/tool;
- expected and actual behavior;
- redacted screenshots or error text;
- service health output if relevant.

Do not include recovery phrases, private keys, Vault passwords, pairing URLs, access tokens, API keys, personal financial data or unredacted wallet activity.

Security vulnerabilities must follow [SECURITY.md](SECURITY.md) and should not be filed publicly.
